Flag Guessing Game

An earlier C# game project that still matters in this portfolio because it highlights both technical progress and how much my development habits improved over time.

Overview

This was a C# game where a player is shown a flag and has to guess the correct country from multiple choices. The project itself is simpler than the others in this portfolio, but it still deserves a place because it represents an earlier stage in my development and the lessons from it stayed with me.

What worked well

The basic gameplay loop worked, and the project helped me think about user interaction, score tracking, and how to structure simple game logic. It also gave me practice collaborating on a shared project, which was still fairly new to me at the time.

Challenges and learning

The biggest issue was not really the code itself. It was the development process. Looking back, I can see how limited our version control habits and team coordination were compared with what I understand now. That is exactly why I still include it.

Why it belongs in this portfolio

It shows growth. Compared with my later projects, this one makes it easier to explain how my thinking changed around planning, collaboration, and code organization over time.

Technologies

C#, Visual Studio, Windows Forms

Key strengths shown

Growth mindset, early problem solving, teamwork lessons, game logic basics
